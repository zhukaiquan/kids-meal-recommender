# Kids Meal Recommender

A minimal React app that helps a parent decide breakfast, lunch, and dinner for one child from a tagged food library.

## Scripts

- `npm install`
- `npm run dev`
- `npm test -- --run`
- `npm run build`

## Deployment

- Production URL: `https://zhukaiquan.github.io/kids-meal-recommender/`
- Pushes to `main` trigger the GitHub Pages workflow in `.github/workflows/deploy.yml`

## Core Behaviors

- Daily breakfast, lunch, and dinner generation
- Per-meal refresh without changing the other meals
- Soft seven-day variety preference
- Browser-local persistence
