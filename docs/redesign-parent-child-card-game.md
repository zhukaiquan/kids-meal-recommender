# 亲子翻牌菜单第一版设计

## 目标

把“今天给孩子吃什么”的决策过程改造成一个轻量亲子小游戏。孩子负责翻开三餐卡牌，家长负责确认和必要时重抽，最终得到今天的早餐、午餐、晚餐。

## 第一版范围

- 首页改为“绘本厨房里的今日菜单翻牌游戏”。
- 早餐、午餐、晚餐是三张默认盖住的卡牌。
- 点击任意卡牌可翻开，不强制顺序。
- 翻开后展示推荐食物、食物图片和一句儿童友好的提示语。
- 每张卡支持“就吃这个”和“再抽一张”。
- 三张卡都确认后展示“今日小厨师任务完成”的完成反馈。
- 食物库和历史记录保留为家长区，不抢首页主流程。

## 不做的内容

- 不做拖拽餐盘。
- 不做复杂养成、积分、等级和长期徽章系统。
- 不改推荐算法和数据结构。
- 不新增需要登录或云端同步的能力。

## 视觉方向

- 主视觉：儿童绘本厨房。
- 关键元素：厨师帽、锅铲、餐盘、胡萝卜、小番茄、小动物助手、星星贴纸。
- 色彩：薄荷绿、番茄红、奶油黄、天空蓝。
- 卡牌：实体游戏卡质感，卡背有手绘图案，翻开后像贴纸相框。

## 实现策略

保留现有推荐、刷新、图片搜索、本地存储和历史记录能力，只重写首页体验层和视觉系统。这样第一版可以尽快上线，同时后续仍能继续迭代更强的游戏机制。
